# MediaPipe Gesture Recognizer Model

Place the `gesture_recognizer.task` model file in this directory for hand gesture recognition.

## Download Instructions

1. Visit: https://developers.google.com/mediapipe/solutions/vision/gesture_recognizer#models
2. Download the `gesture_recognizer.task` file
3. Place it here as `app/src/main/assets/gesture_recognizer.task`

**Note:** The app works without this model — gesture recognition will simply be disabled.
